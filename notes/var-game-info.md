Переменная `$game`
Содержит следующие поля:
```php
$game[
    floc => расположение флага
    fid => ник владельца флага
    fchar => имя владельца флага
    stop => время до остановки(не используется)
    msg => сервисное сообщение
    lastai => время последнего обновления
    journal => сообщение "для всех"(реализовано не до конца)
    info => информация(не используется)
];
```
