# Переменная `$game`

Содержит следующие поля:
```php
$game = [
//   имя          тип         назначение
    'floc'    => 'строка', // расположение флага
    'fid'     => 'строка', // ник владельца флага
    'fchar'   => 'строка', // имя персонажа владельца флага
    'stop'    => 'число',  // время до остановки(реализовано не до конца)
    'msg'     => 'строка', // сервисное сообщение
    'lastai'  => 'число',  // время последнего обновления списка персонажей online
    'journal' => 'строка', // сообщение "для всех"(реализовано не до конца)
    'info'    => 'строка', // информация(не используется)
];
```
